Программа для tg бота, которая узнаёт погоду в данный момент времени


Использованные библиотеки:
1. Сторонние
1.1. requests для получения get-запросов
1.2. aiogram для самого tg бота

2. Встроенные
2.1. datetime для перевода времени в понятный для человека формат

Программа узнаёт основные типы погоды на улице в разных городах света
(Ясно, облачно и т.д.)
Также даёт пользователю информацию о температуре, влажности, давлении
и о продолжительности дня (восход и закат солнца соответственно)
В качестве бонуса добавил различные эмодзи для разных типов погод

По сравнению с BTC price это уже более сложный код, где нужно поработать с json.
Но на деле ничего сложного, можно часик посидеть и разобраться.
